CSS

CSS stands for Cascading Style Sheet. CSS provides the style to HTML elements and website data.

CSS Advantages:-

Give Style to Data Information.
Give style to web Structure.
Saves a lot of time.

In three ways we can add CSS to HTML elements

.

1. External CSS
2. Internal CSS
3. Inline CSS

External CSS:


External CSS means that web page is separate and .CSS file is a separate file but web page acess the .CSS

Internal CSS means that CSS is used inner in of html.

Inline CSS means to put the css code inside the html tags.
